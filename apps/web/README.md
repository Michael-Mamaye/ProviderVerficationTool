# Provider Verification Tool — Web

Next.js (App Router) frontend for the Provider Verification Tool, styled with Tailwind CSS and shadcn/ui.

## Development

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000).

## Environment variables

Copy `.env.example` to `.env.local` and fill in the values:

```bash
cp .env.example .env.local
```
